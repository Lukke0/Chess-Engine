# Chess Engine

A chess engine written in C++ using SFML.

## Current features

- Basic project structure
- SFML setup
- FEN parsing work in progress

## Requirements

- Visual Studio with the **Desktop development with C++** workload
- SFML (included in this repository)

## Build

1. Open `ChessEngine.sln` in Visual Studio.
2. If prompted, allow Visual Studio to retarget the project to your installed toolset.
3. Build the solution (`Ctrl+Shift+B`).
4. Run the project (`F5`).
